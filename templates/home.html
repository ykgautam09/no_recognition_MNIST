HAND WRITTEN DIGIT RECOGNITION SYSTEM

this project 'HAND WRITTEN DIGIT RECOGNITION SYSTEM' solves The MNIST handwritten digit classification problem is a standard dataset used in computer vision and deep learning by using kNeighbourClassifier,lazy learning through MNIST dataset maintained by professor YannLecun at yann.lecun.com .

  • The digit images in the MNIST set were originally selected and experimented with by Chris Burges and Corinna Cortes using bounding-box normalization and centering
    •
  • MNIST dataset basically originated from dataset generated from NIST
  • This project is building block of various system ,as in application of recognising plate no of vehicles,cheque no, etc.
